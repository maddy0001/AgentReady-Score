# AgentReady Score - Frontend

## Setup
```bash
npm install
npm run dev
```

## Build for Production
```bash
npm run build
npm run preview
```

## Components

- **Sidebar** - Navigation + mini score display
- **ScoreDashboard** - Main score with charts
- **SimulationLog** - Real-time SSE feed
- **ActionPlan** - Ranked fixes by revenue
- **BenchmarkIndex** - Competitor comparison

## Tech Stack

- React 18
- Vite 7
- Tailwind CSS
- Recharts (charts)
- Axios (HTTP client)
