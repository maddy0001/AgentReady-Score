# Contributing to AgentReady Score

## Development Setup

1. Fork the repository
2. Clone your fork
3. Create a feature branch
4. Make your changes
5. Test thoroughly
6. Submit a pull request

## Code Style

**Python:**
- Follow PEP 8
- Use type hints
- Add docstrings

**JavaScript:**
- Use ES6+ syntax
- Follow Airbnb style guide
- Use functional components

## Testing
```bash
# Backend tests
python ml/predict.py
python simulator.py

# Frontend (manual)
npm run dev
# Test all 4 screens
```

## Pull Request Process

1. Update README.md if needed
2. Test on both Windows and Mac
3. Include screenshots for UI changes
4. Reference issue number if applicable
```
